Description
~~~~~~~~~~~~~~~
This widget show categories which contains at least one post in "publish" status.
So categories that have only posts in "feature" status will be hidden


Installation
~~~~~~~~~~~~~~~

1) Put plugin folder to your wp plugins folder:
/www/wp-content/plugins/*

2) Go to your admin panel and activate plugin

3) Go to widgets section in admin panel and put widget to the sidebar

4) Enjoy
